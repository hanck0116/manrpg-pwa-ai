ManRPG v18 + 최종 병합 충돌 해결 요약

병합 원칙
- V18은 세계관, 진행 방식, GM 출력 강제 구조의 기본 골격으로 유지한다.
- 최종 파일은 실제 실행 시트 기준이다.
- 같은 항목이 충돌하면 최종 파일의 수치·보상·파생규칙·마법·마법서·특성 처리를 우선한다.

주요 충돌 해결
1. 평타 피해
- V18: floor((힘 + 체력) / 5) + 2
- 최종: floor((힘 + 체력) / 10) + 2
- 적용: 최종 기준 floor((힘 + 체력) / 10) + 2

2. 층 클리어
- V18: 층 클리어 즉시 레벨 +5, 스탯 포인트 +15, 코인 +1, 회복, 상태이상 제거.
- 최종: [층 클리어]는 HP/MP 회복, 상태이상 제거, 층당 1회 특성 사용 플래그 초기화, 자기상환 배수 초기화만 처리한다. 레벨 +5는 별도 [5Lv 플러스] 처리이며, 자동 코인 +1은 적용하지 않는다.
- 적용: 최종 시트 기준.

3. 보상 후보 수
- V18: 보상 2개 생성 후 1개 선택.
- 최종: 외모에 따라 후보 수와 선택 수가 달라진다.
- 적용: 최종 rewardConfig 기준.

4. 보상 풀
- V18: 전승, 마법서, 마나 포션, 체력 포션 중심.
- 최종: 스킬 초기화권, 무공서, 마법서, 추가 코인 +1, 추가 코인 +2 중심. 포션 보상은 제거한다.
- 적용: 최종 rollReward 기준.

5. 무공서
- 최종: 외공서 40%, 내공서 40%, 검기 20%.

6. 마법서 등급 뽑기
- 최종: 기초 50%, 중급 30%, 고급 10%, 멀티캐스팅의 서 9%, 마도서 1%.

7. 마법서 습득 난이도
- 최종: 기초 50, 중급 70, 고급 100, 마도서 100.
- 성공 조건: 1d난이도 < 지혜 또는 지혜 >= 난이도.
- 실패 시 마법서는 유지한다.

8. 마법 위력
- 최종: 마법 위력은 사용 MP를 절대 넘지 않는다. 서클, 범위 유형, 범용성에 따라 개별 위력/MP를 계산한다.

9. 판매가
- 최종 makeItem 기준 적용.
- 외공서 5, 내공서 5, 검기 12, 멀티캐스팅의 서 10, 스킬 초기화권 5, 기초 마법서 3, 중급 마법서 5, 고급 마법서 8, 마도서 15.

10. 기술/전승 처리
- V18의 적 기술 약화판 전승 규칙은 서사 보조로만 유지한다.
- 최종에서는 기술목록을 별도 운용하며, 공법/무공/극기/오리지널 스킬/정령/헤일로/입천/유물/연혼염/빙백연혼 등 출처마다 기술 1개 제작을 기준으로 한다.
