## 스킬 1차 구현 테스트

1. 정비 단계에서 스킬 생성 폼으로 피해/회복/방어 스킬을 생성합니다.
2. 플레이어 메인 턴에서 보유 스킬의 큐 추가 버튼을 눌러 `skillId`가 있는 스킬 행동을 추가합니다.
3. 턴 마무리 후 피해 스킬은 적 HP, 회복 스킬은 플레이어 HP, 방어 스킬은 방어 상태만 변경하는지 확인합니다.
4. 스킬 피해로 적 HP가 0이 되면 `floor-cleared`로 전환되는지 확인합니다.
5. 스킬 초기화권을 사용하면 보유 스킬만 비워지고 플레이어 1명, 적 1명, 보스 없음, 고정 11x11 맵 조건이 유지되는지 확인합니다.
6. AI/Worker 응답이 HP, MP, 피해, 보상, 위치, 스탯, 스킬 효과를 변경하지 않는지 확인합니다.
